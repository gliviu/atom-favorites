favorites
==========
Registers **File->Favorites** menu with useful commands to access favorite files and folders.

**Install**

```apm install favorites```

**Configuration example**

```cson
favorites:
  paths: [
    "topic:Dev"
    "/main.js"
    "/README.md"
    "topic:Other"
    "/notes.txt"
    "/downloads"
  ]

```
